# StockMarketApp
Ionic, Cordova web app that works on iOS and Android mobile devices.

**Start Ionic Project**
```
$ ionic start YourAppsName [template]
```

**View App in Browser**
```
$ ionic serve
```

**Finish Setting Up Gulp and SASS**
```
$ npm install -g gulp
$ npm install -g sass
$ ionic setup sass
$ gulp sass
```

Add the new created stylesheets to index.html
